Angular

Angular est un framework souvent utilisé pour créer des "Single Page Application".

Voici ce que C.R.U.D siginifie

Lettres Définition
C "Create" Créer/ajouter de nouvelles données.
R "Read/Retrieve" lire/récupéer les données.
U "Update" Mettre à jours les données
D "Delete" Effacer les données

Voici les relations uml et ce qu'elles signifient

Formes Définition Code Typescript Remarques
A -----> B Héritage class A extends B B est la classe Parent
A -----⊳ B Implémentation class A implements B B est une interface
A ----- B Association class A utilise B ou l'inverse Il y a 3 types d'associations
A -----◆(noir) B Composition class A est un attribut de class B et doit être instancié dans tout constructeur de B Aucune remarque
A -----◆ B Agrégation class A est un attribut de class B et il n'y a pas besoin de l'instancier dans tout constructeur de B Aucune remarque
